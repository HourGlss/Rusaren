# client

This directory contains client-facing code and assets for the Godot shell and any supporting client-side tooling.
Use the structure notes below to find the right file or subfolder quickly.

## Structure
- `godot/`: the Godot 4 browser shell that talks to the Rust backend over websocket signaling and WebRTC.
- `scripts/`: reserved space for future standalone client-side helper scripts. It is currently an intentionally empty placeholder.
- `README.md`: This guide documents the folder structure and explains what the checked-in files and subfolders are for.
