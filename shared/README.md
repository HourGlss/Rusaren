# shared

This directory contains cross-cutting design notes, content planning, and protocol documentation that sit above individual crates.
Use the structure notes below to find the right file or subfolder quickly.

## Structure
- `content/`: reserved space for shared authored-content notes outside the live runtime content tree.
- `docs/`: long-form project documentation covering architecture, gameplay, networking, testing, and operations.
- `protocol/`: reserved space for standalone protocol documents or generated references outside the crate-level docs. It is currently an intentionally empty placeholder.
- `README.md`: This guide documents the folder structure and explains what the checked-in files and subfolders are for.
