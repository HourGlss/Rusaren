# content

This directory contains reserved space for shared authored-content notes outside the live runtime content tree.
Use the structure notes below to find the right file or subfolder quickly.

## Structure
- `abilities/`: reserved notes for shared ability concepts and authored-content planning. It is currently an intentionally empty placeholder.
- `classes/`: reserved notes for shared class-content planning outside the live runtime YAML files. It is currently an intentionally empty placeholder.
- `maps/`: reserved notes for shared map-content planning outside the live runtime ASCII maps. It is currently an intentionally empty placeholder.
- `README.md`: This guide documents the folder structure and explains what the checked-in files and subfolders are for.
