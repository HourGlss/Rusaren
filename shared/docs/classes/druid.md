# Class: Druid (WoW Druid / EQ nature caster)

Identity:
- Flexible nature skirmisher that blends mobility, roots, healing, and chill pressure.
- Comfortable playing around terrain and cover instead of raw lane dominance.

Passive identity:
- `Wildshape Instinct`: the druid thrives in cluttered terrain and extended skirmishes.
- This is current design intent only; runtime passives are not a separate system yet.

Current authored combat profile:
- thorn projectile poke
- pounce dash
- healing pulse with HoT
- root burst
- chill beam finisher
