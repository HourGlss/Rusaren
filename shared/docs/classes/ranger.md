# Class: Ranger (WoW Hunter / EQ Ranger)

Identity:
- Long-lane marksman who controls space with precise arrows and roots.
- Trades raw support power for angle control and ranged pick threat.

Passive identity:
- `Hunter's Focus`: the ranger is rewarded for clean sightlines and disciplined spacing.
- This is documented class identity for now, not a separate runtime passive mechanic yet.

Current authored combat profile:
- long-range projectile poke
- disengage dash
- targeted root burst
- thin high-range beam
- wide ranged barrage for cracking fortified positions
