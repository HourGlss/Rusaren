# Class: Necromancer (WoW warlock-ish / EQ necro attrition caster)

Identity:
- Attrition caster that wins through poison, roots, silence, and fight erosion.
- Prefers drawn-out engagements and denial over immediate burst racing.

Passive identity:
- `Grave Pact`: the necromancer gets stronger as the fight drags on and statuses accumulate.
- This is authored class identity for later passive support, not an active runtime passive yet.

Current authored combat profile:
- poison projectile
- root burst
- dark sustain pulse
- silence beam
- poison nova closer
