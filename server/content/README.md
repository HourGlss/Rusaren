# content

This directory contains runtime-authored gameplay content loaded by the backend at startup.
Use the structure notes below to find the right file or subfolder quickly.

## Structure
- `maps/`: ASCII arena map sources used by the live backend.
- `mechanics/`: authored mechanic-family registry data that validates and classifies gameplay content.
- `skills/`: authored class and skill definitions shipped with the current backend build.
- `README.md`: This guide documents the folder structure and explains what the checked-in files and subfolders are for.
