# input_frame_roundtrip

This directory contains checked-in seed inputs for the `input_frame_roundtrip` fuzz target.
Use the structure notes below to find the right file or subfolder quickly.

## Structure
- `Hash-named files`: Each SHA-like file is a minimized fuzz seed kept exactly as emitted by the fuzzing toolchain so replay coverage stays stable for `input_frame_roundtrip`.
- `README.md`: This guide documents the folder structure and explains what the checked-in files and subfolders are for.
