# static

This directory contains checked-in static asset stubs and the landing place for the exported web client bundle.
Use the structure notes below to find the right file or subfolder quickly.

## Structure
- `webclient/`: ignored web-export output for the Godot client. It is populated locally or in CI before deploys.
- `README.md`: This guide documents the folder structure and explains what the checked-in files and subfolders are for.
